Netflix Content Analysis Dashboard
About the Project

This project is a Netflix Content Analysis Dashboard created using Power BI.
The main goal of this project is to analyze Netflix movies and TV shows and understand content trends using interactive visualizations.

I worked on data cleaning, data analysis, and dashboard design to make the report simple, interactive, and easy to understand.

What I Analyzed
Total Netflix Titles
Movies vs TV Shows
Content Released Over the Years
Most Popular Genres
Top Content Producing Countries
Content Ratings Distribution
Tools Used
Power BI
Power Query
CSV Dataset
Data Cleaning

Before creating the dashboard, the dataset was cleaned in Power Query by:

Removing null values
Fixing column names
Changing data types
Removing unnecessary columns
Cleaning country and genre values
Dashboard Features
Interactive Filters
KPI Cards
Line Chart for Release Trends
Pie Chart for Content Distribution
Genre Analysis
Country-wise Content Analysis
Ratings Distribution
Key Insights
Movies are more available than TV Shows on Netflix
Netflix content increased rapidly after 2015
Drama and International Movies are among the most popular genres
Some countries contribute more content compared to others
