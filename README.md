\# Netflix Content Analysis Dashboard



\## About the Project



The Netflix Content Analysis Dashboard is a Power BI project created to analyze movies and TV shows available on Netflix using interactive visualizations and data analysis techniques.



The main objective of this project is to understand Netflix content trends, identify popular genres, analyze release patterns, study country-wise content production, and compare Movies vs TV Shows.



This project was created using real-world Netflix data and helped improve my practical knowledge in data cleaning, dashboard design, and business insights generation using Power BI.



The dashboard was designed to be simple, interactive, visually attractive, and easy to understand.



\---



\# Project Objectives



\- Analyze Netflix content trends

\- Compare Movies and TV Shows distribution

\- Study release trends over the years

\- Identify popular genres on Netflix

\- Analyze country-wise content production

\- Understand ratings distribution

\- Build an interactive dashboard using Power BI



\---



\# Tools \& Technologies Used



\- Power BI

\- Power Query

\- CSV Dataset



\---



\# Dataset Information



The dataset contains Netflix content information such as:



\- Show Title

\- Type (Movie / TV Show)

\- Genre

\- Country

\- Rating

\- Release Year

\- Director

\- Cast

\- Duration



The dataset was used to perform analysis and create interactive dashboard visualizations.



\---



\# Data Cleaning Process



Before building the dashboard, the dataset was cleaned and transformed using Power Query inside Power BI.



The cleaning process included:



\- Removing null values

\- Fixing incorrect column names

\- Changing data types

\- Removing unnecessary columns

\- Cleaning genre and country values

\- Preparing the dataset for visualization



Data cleaning improved the quality and accuracy of the dashboard analysis.



\---



\# Dashboard Features



The dashboard contains multiple interactive visuals and filters for better analysis.



\### Main Features



\- KPI cards

\- Interactive filters

\- Line chart for release trends

\- Pie chart for Movies vs TV Shows

\- Genre analysis

\- Country-wise content analysis

\- Ratings distribution analysis



\---



\# Analysis Performed



\## Content Distribution Analysis



Compared the number of Movies and TV Shows available on Netflix.



\## Release Trend Analysis



Analyzed how Netflix content increased over the years and identified growth trends.



\## Genre Analysis



Studied the most popular genres available on Netflix.



\## Country Analysis



Identified countries producing higher amounts of Netflix content.



\## Ratings Analysis



Analyzed content ratings distribution across Netflix titles.



\---



\# Key Insights



\- Movies are more available on Netflix compared to TV Shows

\- Netflix content increased rapidly after 2015

\- Drama and International Movies are among the most popular genres

\- Some countries contribute significantly more content

\- TV Shows have grown steadily in recent years
